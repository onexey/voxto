# Preferences

Voxto groups settings into dedicated tabs:

- **General** — model selection, hotkey behavior, startup, and update preferences.
- **Markdown files** — enable the output and choose the folder for per-recording `.md` files.
- **Todo list** — enable the output and choose the shared Markdown todo file.
- **Cursor location** — enable direct insertion and optionally press Enter after the text is sent.
- **About** — version and repository links.

Output tabs store their configuration independently inside the main settings file. That keeps one output's preferences isolated from the others and makes future output settings changes safer.
